## About the Project

**StockVision** is an AI-powered platform that predicts future stock movements using deep learning.  
It combines a **TensorFlow LSTM neural network** with a modern **Next.js frontend**, offering users real-time insights and buy/sell recommendations based on stock market data from **Yahoo Finance**.

💡 The goal:  
To build a full-stack, production-ready AI project that demonstrates **data science**, **machine learning**, and **web development** expertise.

---

## 🚀 Features

- 📊 Fetches real-time and historical stock data with **yfinance**
- 🧠 Predicts stock price trends using an **LSTM model (TensorFlow/Keras)**
- ⚙️ Interactive **FastAPI backend** for live predictions
- 💻 Modern **Next.js frontend** with responsive UI
- 🔍 Visualizes stock charts and model confidence
- 🟢 Provides **Buy / Sell / Hold** recommendations

---

## 🧩 Tech Stack

### **Machine Learning**
- TensorFlow / Keras – LSTM network for time series prediction  
- Pandas, NumPy – Data handling & feature scaling  
- yFinance – Fetching financial data  
- Matplotlib, Seaborn – Visual exploration  
- Scikit-learn – Metrics & preprocessing  

### **Backend**
- FastAPI – Lightweight API for serving predictions  
- Uvicorn – ASGI server  

### **Frontend**
- Next.js (React + TypeScript) – Modern web framework  
- TailwindCSS – UI styling  
- Axios / SWR – API data fetching
